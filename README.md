# ✨ Benz-BMW Shine

Marketing site for a premium automotive detailing service specialising in Mercedes-Benz and BMW care.

🔗 Live: _(link once deployed)_

---

## Stack

| Layer | Tech |
| :--- | :--- |
| Framework | Vite · React · TypeScript |
| Edge runtime | Cloudflare Workers |
| Styling | Tailwind · shadcn/ui · Radix primitives |
| Forms | React Hook Form · Zod |
| Package manager | Bun |

---

## Features

- Service menu (exterior / interior / paint correction / ceramic coating)
- Before/after gallery
- Booking request form
- WhatsApp CTA

---

## Local development

```bash
bun install
bun run dev
```

### Deploy to Cloudflare

```bash
bun run build
wrangler deploy
```
